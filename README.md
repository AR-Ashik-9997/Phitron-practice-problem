# Phitron Practice Problem

A comprehensive collection of practice problems and challenges for data science and machine learning from Phitron.

## 📁 Contents

- **Beginner_Friendly_Practice_Set.ipynb** - Introductory exercises covering:
  - Exploratory Data Analysis (EDA)
  - Data preprocessing
  - Missing value handling
  - Statistical analysis
  - Data visualization

- **Eid_Vacation_Practice_Challenge.ipynb** - Advanced challenges including:
  - Real-world dataset analysis (Adult Income Dataset)
  - Feature engineering
  - Data normalization and scaling
  - Machine learning pipeline construction
  - Model evaluation

## 🛠️ Requirements

- Python 3.7+
- Virtual Environment (recommended)

### Dependencies

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn

## 📦 Installation

### 1. Create and Activate Virtual Environment

**On Windows (PowerShell):**

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

**On macOS/Linux:**

```bash
python -m venv .venv
source .venv/bin/activate
```

### 2. Install Required Packages

```bash
pip install -r requirements.txt
```

## 🚀 Getting Started

1. Ensure your virtual environment is activated
2. Open either notebook in Jupyter/VS Code
3. Follow the instructions in each cell
4. Run cells using `Shift + Enter`
